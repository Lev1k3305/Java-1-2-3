# Сеть кибер-детекторов
**Практическая работа №8 по дисциплине «Многопоточное и распределённое программирование»**  
Декабрь 2025

## Сюжет

Мегаполис охвачен вирусными ИИ. Чтобы обнаружить и уничтожить угрозу, вы разрабатываете эволюционирующую сеть кибер-детекторов:  
от примитивных независимых дронов → до полностью синхронизированной распределённой вычислительной системы, подключённой к Центральному Узлу.

## Задача

Обработать массив из **100 000** случайных целых чисел (0–10 000) с помощью 20 детекторов, последовательно усложняя архитектуру многопоточной обработки.

## Этапы эволюции сети

| Этап | Название                          | Технологии Java                              | Результат |
|------|-----------------------------------|---------------------------------------------------|---------|
| 1    | Ручные потоки                     | `Thread`, `Runnable`, `join()`                    | Локальные суммы по сегментам |
| 2    | Синхронизация                     | `AtomicLong` / `synchronized`                     | Корректная глобальная сумма |
| 3    | Подключение к Центральному Узлу   | `ExecutorService`, `Callable`, `Future`           | Централизованный сбор результатов |
| 4    | Обнаружение аномалий              | Класс `Result { long sum; int anomalies; }`       | Подсчёт чисел > 9000 |
| 5    | Обработка по готовности           | `ExecutorCompletionService`                       | Динамический вывод отчётов |
| 6    | Фазированная координация          | `CyclicBarrier`                                   | Синхронизация фаз анализа |
| 7    | Анализ производительности         | Замеры времени                                    | Сравнительная таблица |
