# Мини-RPG: Битва в подземелье  
**Практическая работа №7** — ООП, полиморфизм и паттерны проектирования  
Декабрь 2025

## Описание задания

Герой спускается в опасное подземелье, сражается с монстрами (гоблины, скелеты, дракон), собирает и использует предметы, принимает тактические решения.  
Программа демонстрирует применение полиморфизма, паттернов проектирования, собственных исключений и коллекций.

## Реализованные паттерны и требования

| Паттерн / Требование                  | Где используется                                            | Баллы |
|---------------------------------------|-------------------------------------------------------------|-------|
| Factory Method                        | Создание героев, монстров, предметов                        | 7     |
| Strategy                              | Разные типы атак (ближний бой, магия, стрельба)             | 5     |
| Decorator                             | Усиленные предметы (`GreaterPotion`, `FireSword` и др.)     | 10    |
| Observer                              | Логгер следит за HP, атакой и защитой героя                 | 8     |
| Template Method                       | Пошаговый алгоритм боя                                      | 7     |
| Memento                               | Сохранение/восстановление состояния перед боем с драконом   | 9     |
| Собственные исключения                | `LowHealthException`, `InventoryFullException`, `DragonFireException` | +5 |
| Коллекции + итераторы                 | Рюкзак (`List<Item>`), рейтинг игроков (`Map<String,Integer>`), `Iterator` | +5 |
| Наследование и полиморфизм            | Базовые классы `Hero`/`Monster`, переопределение методов    | +5 |
| **Итого**                             |                                                             | **61 → 100** |
